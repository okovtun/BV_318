Array - это набор переменных одного типа в непрерывной области памяти.
Переменные массива называют его элементами.

			type name[SIZE];
type - тип элементов массива;
name - имя массива;
SIZE - количество элементов массива, может быть задано только целочисленным константным значением;

			const int SIZE = 5;
			int arr[SIZE];
